# SkyproWallet

SkyproWallet — SPA для учета и анализа личных расходов с визуализацией статистики и управлением транзакциями.

Приложение позволяет зарегистрироваться, вести учет транзакций, анализировать расходы за выбранный период и получать наглядную статистику по категориям через удобный адаптивный интерфейс.

## Установка проекта

Клонируйте репозиторий:

```bash
git clone https://github.com/kors-ak/skypro-wallet.git
```

Перейдите в папку проекта:

```bash
cd skypro-wallet
```

Установите зависимости:

```bash
npm install
```

Запустите проект:

```bash
npm run dev
```

Сборка проекта:

```bash
npm run build
```

## Организация работы

- Управление задачами велось в Kaiten.
- Основной веткой разработки являлась `dev`.
- Каждый участник работал в собственной Git-ветке.
- Все изменения проходили code review.
- После проверки изменения объединялись в ветку `dev`.
- После завершения разработки ветка `dev` объединялась с `main`.
